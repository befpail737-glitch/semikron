# SEO/GEO Claude Skills Library

## Overview

This is a comprehensive skill library for executing professional SEO and GEO (Geographic Optimization) analysis for global distributor businesses.

## Components

### 1. keyword-research
- Long-tail keyword discovery (30+ keywords)
- 6-category classification system
- Search volume and competition analysis
- Priority scoring

### 2. serp-analysis
- Search Engine Results Page analysis
- Featured snippets identification
- PAA (People Also Ask) extraction
- Local pack analysis

### 3. competitor-analysis
- Top 10 competitor profiling
- Title structure analysis
- Trust signals evaluation
- Localization assessment
- E-E-A-T scoring

### 4. content-quality-auditor
- CORE-EEAT framework audit
- CITE trust scoring system
- Content gap analysis
- Quality recommendations

### 5. performance-reporter
- 10-chapter report generation
- Markdown output format
- Actionable recommendations
- Timeline and milestones

## Contract Mechanism

Each component follows a strict contract:
1. **Input**: Project context, target countries, core keywords
2. **Process**: Research → Analysis → Audit → Report
3. **Output**: Structured data + recommendations

## Usage

```
使用 seo-geo-claude-skills 执行 [component]
项目：[project details]
目标：[objectives]
```
