FAE Technical Commentary

Expert insights on rectifier selection

For high-temperature applications, we recommend SEMIPONT modules with enhanced thermal design that maintains consistent forward voltage characteristics across extended temperature ranges. The bolt diode technology offers superior thermal performance compared to traditional packages.

When designing for high-repetitive surge currents, consider the latest SEMIPONT generation which features improved surge current handling capability. This is particularly important in welding and power supply applications where load variations are common.

For applications with inductive loads, select modules with appropriate reverse recovery characteristics. The bolt diode configuration provides excellent thermal management, but ensure proper mounting torque for optimal heat dissipation.
