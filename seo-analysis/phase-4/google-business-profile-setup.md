## 🔗 GBP 与网站整合

### 网站标记要求
每个国家页面需要添加 LocalBusiness Schema:
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "LiTong Group - [Country]",
  "@id": "https://semikron.elec-distributor.com/distributor/[country]/",
  "url": "https://semikron.elec-distributor.com/distributor/[country]/",
  "telephone": "+[country-code]-xxx-xxx-xxxx",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "[Street Address]",
    "addressLocality": "[City]",
    "postalCode": "[Postal Code]",
    "addressCountry": "[Country Code]"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": XX.XXXX,
    "longitude": XX.XXXX
  },
  "openingHoursSpecification": {
    "@type": "OpeningHoursSpecification",
    "dayOfWeek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"],
    "opens": "09:00",
    "closes": "17:00"
  }
}
```

### 网站链接要求
- 在 GBP 中添加对应国家页面链接
- 确保网站上有完整的联系信息
- 添加 Google Maps 嵌入 (如适用)

## ⚠️ 重要注意事项

1. **地址要求**: 每个 GBP 需要一个真实的实体地址，不能使用虚拟办公室
2. **电话号码**: 建议使用当地国家的本地电话号码
3. **验证方式**: 大多数国家支持明信片验证，部分支持电话/邮件验证
4. **语言一致性**: GBP 信息语言应与目标国家页面语言一致
5. **定期更新**: 至少每月更新一次照片和 Posts
