## 🔧 工具设置

### 1. Google Analytics 4 (GA4)

#### 设置步骤
1. 创建 GA4 属性
2. 安装跟踪代码到所有页面
3. 设置事件跟踪:
   - 表单提交 (contact, quote)
   - 文件下载 (PDF guides)
   - 出站链接点击
   - 电话点击
4. 设置转化目标:
   - 询盘提交
   - 电话拨打
   - 邮件点击
   - PDF下载

#### 自定义报告
- 按国家/地区划分的有机流量
- 着陆页表现报告
- 转化路径分析
- 设备类型分析

### 2. Google Search Console (GSC)

#### 设置步骤
1. 验证网站所有权 (DNS/HTML文件)
2. 提交 XML Sitemap
3. 设置国际定位 (如果适用)
4. 添加所有子域名/变体

#### 监控重点
- 搜索查询报告
- 页面体验报告
- Core Web Vitals 报告
- 索引覆盖率报告
- 移动可用性报告

### 3. 排名跟踪工具

#### 推荐工具
- **Ahrefs** - 全面的SEO分析
- **SEMrush** - 关键词研究和排名跟踪
- **SERPWatcher** - 日常排名监控

#### 设置清单
- [ ] 添加所有目标关键词 (48个)
- [ ] 设置竞争对手跟踪
- [ ] 配置排名下降警报
- [ ] 设置每周/每月报告

### 4. 本地SEO工具

#### Google Business Profile
- 监控所有13个国家的GBP
- 设置评价提醒
- 跟踪照片和Posts表现

#### 本地排名跟踪
- **Local Falcon** - 网格化本地排名跟踪
- **BrightLocal** - 综合本地SEO报告

## 🚨 警报设置

### 关键警报

| 警报类型 | 阈值 | 通知方式 |
|---------|------|---------|
| 有机流量下降 | > 20% | 邮件 |
| 关键词排名下降 | > 10位 | 邮件 |
| 页面索引问题 | > 5页 | 邮件 |

| 服务器错误 (5xx) | > 10 | 邮件 |
